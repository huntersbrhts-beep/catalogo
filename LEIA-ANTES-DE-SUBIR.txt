DU LANCHES - V7 CORREÇÕES

Alterações desta versão:
- ADM agora permite excluir categoria do catálogo.
- Carrinho pergunta forma de pagamento.
- Campo de troco aparece quando pagamento é Dinheiro.
- Telefone é salvo no histórico local para programa de fidelidade.
- Fidelidade mostra brinde a partir de 3 pedidos e desconto a partir de 5/10 pedidos.
- Produtos podem ser marcados como Disponível ou Indisponível no ADM.
- Rodapé mostra redes sociais configuráveis e texto "Desde 2019".
- Prêmios da roleta agora podem ser editados, salvos e excluídos no ADM.
- Removida a mensagem de versão do topo do site.
- Cache atualizado para GitHub Pages.

Como subir no GitHub Pages:
1. Extraia o ZIP.
2. Envie o index.html e a pasta assets na raiz do repositório.
3. Não envie a pasta inteira como subpasta.
4. Depois de publicar, use Ctrl + F5 no navegador.

Atenção:
As configurações visuais, redes sociais, cupons, roleta e fidelidade continuam salvas no navegador pelo localStorage, seguindo o padrão das versões anteriores.
Se você alterar no ADM usando um navegador, a alteração aparece nesse mesmo navegador. Para salvar essas configurações para todos os aparelhos, o próximo passo é criar uma tabela de configurações no Supabase.
